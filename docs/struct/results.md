# STResults

## Constructor
```js
constructor(/*Object*/ stats, /*Array*/ results)
```
### Arguments
1. `/*Object*/ stats`
	* The object to get [first](#first), [last](#last), and [count](#count) from. Same types.
2. `/*Array*/ results`
	* An array of the actual search results.

### last
Number<br/>
