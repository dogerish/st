# STStats
Structure for holding player statistics.

## Constructor
```js
constructor(/*Object*/ from)
```
### Arguments
1. `/*Object*/ from`
	* The object to get properties from. Literally just copies the properties - same keys, same types.

## Properties

Number<br/>

Number<br/>
Deaths.
### tks
Number<br/>
Team-kills.
### kpd
Number<br/>
Kills per deaths.
### acc
Number<br/>
